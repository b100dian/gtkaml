resolve

Description:

public virtual MarkupTag? resolve (MarkupResolver resolver) throws ParseError

Called when Gtkaml is resolving. Here replacements in the Gtkaml AST can be made (e.g. MarkupUnresolvedTag -> MarkupTemp). Tags to remove must return 'null' here so that the SymbolResolver can remove them later
